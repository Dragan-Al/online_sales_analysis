# online_sales_analysis
Assignment_Final_Project_Rad sa GitHub i Python

# Opis projekta 

ovo je python projekat detaljno objasnjen u zadatku Assignment_Final_Project_Rad sa GitHub i Python

## klase

Klasa Product 
u klasi se nalaze proizvod sa atributima name, price, quantity 
kao i metode koje prikazuju informacije o proizvodu i 
metoda koja menja kolicinu proizvoda

klasa ProductManager
upravlja listom proizvoda koji su dostupni
Metode:
dodaje proizvod u inventar
prikazuje sve proizvode
racuna ukupnu vrednost inventara
uklanja prema imenu

klasa Cart
Korpa kupaca
metode:
dodaje proizvode u korpu
racuna ukupnu vrednost
prikazuje proizvode u korpi

## Funkcionalnosti
Dodavanje, prikaz i uklanjanje proizvoda iz inventara.
Praćenje količina proizvoda i ukupne vrednosti inventara.
Dodavanje proizvoda u korpu i računanje ukupnog iznosa za naplatu.
Ignorisanje poverljivih fajlova kao što su `config.json` i snimci ekrana preko `.gitignore`.
